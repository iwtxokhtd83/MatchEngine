# API Reference

## Engine

The `Engine` is the main entry point. It manages order books for multiple symbols and coordinates matching.

### Create Engine

```go
import "github.com/iwtxokhtd83/MatchEngine/pkg/engine"

e := engine.New()
```

---

### SubmitOrder

```go
func (e *Engine) SubmitOrder(symbol string, order *model.Order) ([]model.Trade, error)
```

Submits an order to the engine. The engine will attempt to match it against the opposite side of the order book for the given symbol.

**Parameters:**
- `symbol` — trading pair identifier (e.g., `"BTC/USD"`, `"AAPL"`)
- `order` — the order to submit

**Returns:**
- `[]model.Trade` — slice of trades generated by this order (may be empty)
- `error` — non-nil if the order is invalid

**Errors:**
- `order cannot be nil`
- `order quantity must be positive`
- `limit order price must be positive`

**Example:**

```go
order := model.NewLimitOrder("buy-1", model.Buy, decimal.NewFromInt(100), decimal.NewFromInt(10))
trades, err := e.SubmitOrder("AAPL", order)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Generated %d trades\n", len(trades))
```

---

### CancelOrder

```go
func (e *Engine) CancelOrder(symbol, orderID string) bool
```

Removes a resting order from the book.

**Returns:** `true` if the order was found and removed, `false` otherwise.

**Example:**

```go
cancelled := e.CancelOrder("AAPL", "buy-1")
if !cancelled {
    fmt.Println("Order not found or already filled")
}
```

---

### GetTrades

```go
func (e *Engine) GetTrades() []model.Trade
```

Returns a copy of all executed trades across all symbols.

**Example:**

```go
for _, t := range e.GetTrades() {
    fmt.Printf("%s <-> %s | %s @ %s\n",
        t.BuyOrderID, t.SellOrderID, t.Quantity.StringFixed(4), t.Price.StringFixed(2))
}
```

---

### GetOrderBook

```go
func (e *Engine) GetOrderBook(symbol string) *orderbook.OrderBook
```

Returns the order book for a symbol, or `nil` if no orders have been submitted for that symbol.

**Example:**

```go
book := e.GetOrderBook("BTC/USD")
if book != nil {
    fmt.Printf("Best bid: %s\n", book.BestBid().Price.StringFixed(2))
    fmt.Printf("Best ask: %s\n", book.BestAsk().Price.StringFixed(2))
    fmt.Printf("Spread: %s\n", book.Spread().StringFixed(2))
}
```

## Model Constructors

### NewLimitOrder

```go
func NewLimitOrder(id string, side Side, price, quantity decimal.Decimal) *Order
```

Creates a limit order. `Remaining` is set equal to `Quantity`. `Timestamp` is set to `time.Now()`.

### NewMarketOrder

```go
func NewMarketOrder(id string, side Side, quantity decimal.Decimal) *Order
```

Creates a market order. `Price` is zero (ignored during matching). `Remaining` is set equal to `Quantity`.

> All price and quantity fields use [`shopspring/decimal`](https://github.com/shopspring/decimal) for exact decimal arithmetic, avoiding float64 rounding errors in financial calculations.
